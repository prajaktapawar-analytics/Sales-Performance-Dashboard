# 📊 Sales Performance Dashboard

An end-to-end **Data Analytics** project designed to analyze retail sales performance using **Python, SQL, Excel, and Power BI**. This project transforms raw sales data into meaningful business insights through data cleaning, SQL analysis, KPI reporting, and interactive dashboards.

---

## 📌 Project Overview

The Sales Performance Dashboard helps businesses understand their sales trends, customer behavior, product performance, and profitability. It demonstrates the complete analytics workflow from raw data preparation to business intelligence reporting.

---

## 🎯 Project Objectives

- Clean and preprocess raw sales data.
- Store structured data in an SQL database.
- Analyze business performance using SQL queries.
- Calculate important business KPIs.
- Generate actionable business insights.
- Create interactive dashboards using Power BI.
- Document the complete analytics process.

## 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/USERNAME/Sales-Performance-Dashboard.git
```

### Navigate to Project

```bash
cd Sales-Performance-Dashboard
```

### Install Python Dependencies

```bash
pip install pandas numpy matplotlib sqlite3
```

### Run Data Cleaning

```bash
python Dataset/clean_data.py
```

### Populate Database

```bash
python SQL/populate_db.py
```

### Generate Business Insights

```bash
python SQL/generate_insights.py
```
