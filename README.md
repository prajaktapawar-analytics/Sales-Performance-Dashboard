# 📊 Sales Performance Dashboard

An end-to-end **Sales Analytics** project built using **Python, SQL, Excel, and Power BI** to analyze sales performance, customer behavior, product profitability, and business KPIs. The project demonstrates the complete analytics workflow from data preparation to interactive dashboard creation.

---

## 📌 Project Overview

This project analyzes retail sales data to uncover valuable business insights, including sales trends, regional performance, customer segmentation, and product profitability through interactive dashboards and reports.

---

## 🚀 Technologies Used

- Python
- SQL (SQLite)
- Microsoft Excel
- Power BI
- Git & GitHub

---

## 📊 Key Features

- Data Cleaning & Preprocessing
- Sales Trend Analysis
- Customer Segmentation
- Product Performance Analysis
- Profit Analysis
- SQL Data Analysis
- Interactive Power BI Dashboard
- Business KPI Reporting

---

## 📈 Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Monthly Sales Trend
- Regional Sales Performance
- Top Selling Products
- Profit Margin
